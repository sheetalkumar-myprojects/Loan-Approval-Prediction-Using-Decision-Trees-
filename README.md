# Loan-Approval-Prediction-Using-Decision-Trees-
Machine Learning–based loan approval prediction system using a Decision Tree classifier.
This project implements a Machine Learning–based loan approval prediction system using a Decision Tree classifier. The objective is to model a real-world financial decision-making scenario by predicting whether a loan application should be approved or rejected based on applicant features such as income, credit score, loan amount, employment status, existing debt, and marital status. The project demonstrates an end-to-end supervised learning workflow including data preprocessing (categorical encoding and train-test split), model training with hyperparameter tuning to control overfitting, performance evaluation using accuracy, confusion matrix, precision, and recall metrics, and interpretation through feature importance analysis. By leveraging the interpretability of Decision Trees, the model provides transparent, rule-based decision logic, making it suitable for business applications where explainability and accountability are critical.
